# UI and Interaction Baseline

## 1. 导航

左侧测试树包含：

- 基本功能
  - 8.1 测试仪表基本功能验证
- 基站射频指标
  - 8.2 输出功率
  - 8.3 瞬态周期 / OFF 功率
  - 8.4 调制质量
  - 8.5 ACLR
- 基站接收机性能
  - 8.6 参考灵敏度
  - 8.7 ACS (邻道选择性)
- 基站协议一致性
  - 8.8 数据速率与时延
  - 8.9 DFT-s-OFDM 符号结构
  - 8.10 小移频信号接收
  - 8.11 CBRA 随机接入

点击叶子节点后：

1. 重置当前运行状态。
2. 切换当前用例。
3. 更新参数模板。
4. 切回参数配置 Tab。
5. 更新运行表和结果表模板。

## 2. 工具栏

- 连接管理
- 启动后端
- 停止后端
- 配置导入
- 配置导出
- 生成测试报告
- 发送命令
- 后端状态徽标

当前后端启动/停止为界面模拟状态，不代表真实进程生命周期。

## 3. 测试运行

主要控件：

- 当前参数摘要。
- 进度条。
- 开始测试。
- 暂停。
- 停止。
- 两个运行图表容器。
- 运行结果表。
- 导出 CSV、保存截图、查看历史。

当前生命周期事实：

- 开始按钮直接调用当前用例函数。
- 暂停按钮只更新状态栏，没有暂停执行。
- 停止按钮发送仪表停止/关闭命令，但无法可靠中断主线程中的同步等待。
- 8.6、8.7 使用动态“确认 BLER”按钮。

## 4. 结果显示

- 顶部显示 WAIT/PASS/FAIL 及统计摘要。
- 两个结果图表容器会按用例反复替换内容。
- 结果表列随用例变化。
- 导出按钮复用 CSV 导出。

## 5. 日志

- 表列：时间戳、级别、来源、内容。
- 支持等级过滤。
- 支持清空与 CSV 导出。
- ERROR/FAIL、WARN、DEBUG、PASS、INFO 使用不同前景色和背景色。

## 6. 连接管理

展示四类设备：

- 4071 频谱分析仪。
- AIoT 基站。
- 1466 信号发生器。
- 标签模拟器/UART。

当前实际传回主窗口并建立连接的只有 4071 与 1466 IP/端口。

当前“检测全部连接”和“保存预设”主要为提示或模拟行为。
