# Structured Behavior Baseline

## 1. 应用外壳

- 启动窗口标题：`AIoT 测试仪表上位机`
- 初始窗口尺寸：`1500 x 920`
- 最小窗口尺寸：`1280 x 760`
- 初始当前用例：`8.5 ACLR`
- 主 Tab：
  - 参数配置
  - 测试运行
  - 结果显示
  - 测试日志
- 日志等级：
  - 全部
  - DEBUG
  - INFO
  - WARN
  - ERROR
  - PASS
  - FAIL
- 默认频点映射：
  - 200 kHz -> 925 MHz
  - 400 kHz -> 925 MHz
  - 600 kHz -> 925 MHz
  - 800 kHz -> 925 MHz

## 2. 公共参数区域

- 预设：默认配置、高灵敏度、大功率测试。
- 射频参数：载波频率、信道带宽、发射功率、CW 频点、CW 功率、CW 启用。
- 接收机参数：参考信道、D2R 起始功率、终止功率、步进。
- 协议参数：Access Type、AO 编号、FRI k 掩码、标签数量。
- 结果导出选项：CSV、PNG、MAT。

## 3. 8.1 基本功能

### 运行表

- 列：设备、连接/返回、检查项、判定。
- 初始行数：4。
- 检查项：
  - 4071 `*IDN?`
  - 1466 `*IDN?`
  - 4071 `:SYSTem:ERRor?`
  - 1466 `:SYSTem:ERRor?`

### 结果表

- 列：检查项、4071、1466、判定。
- 初始行数：2。
- 汇总项：设备识别、通信方式。

### 当前行为

- 双仪表均连接时执行双仪表标识与错误队列流程。
- 未满足双仪表路径时保留现有单仪表兼容流程。
- 频率和带宽摘要显示 `---`。
- 现有实现开始时直接把进度置为 100%。

## 4. 8.2 输出功率

### 固定参数

- 允许输入功率范围：0 至 15 dBm。
- 分析仪保护上限：28 dBm。
- 功率误差门限：3.01 dB。
- 每个功率点采样次数：10。
- 采样间隔：100 ms。
- 频率误差门限：5 kHz。

### 参数

- 动态功率点输入框仅在 8.2 显示。
- 输入支持逗号、空格和分号分隔。
- 当前界面要求 5 个待测功率点。

### 真实仪表路径

- 1466 配置 CW 频率、功率和 RF 输出。
- 4071 配置频谱并读取 Marker 峰值频率和功率。
- 每点计算平均功率、平均频率、功率误差和频率误差。
- 测试结束停止 4071 测量并关闭 1466 RF。

### 回退路径

- 未同时连接 1466 和 4071 时，保留当前单仪表/模拟兼容行为。

## 5. 8.3 瞬态周期 / OFF 功率

- 带宽：200、400、600、800 kHz。
- SCS：15 kHz。
- 运行表列：带宽/SCS、瞬态周期、OFF 功率、判定。
- 结果表列：带宽、SCS、时间分辨率、OFF->ON、ON->OFF / OFF 功率、判定。
- 本轮只保持当前行为，不扩展业务实现。

## 6. 8.4 调制质量

- 带宽：200、400、600、800 kHz。
- SCS：15 kHz。
- 运行表列：带宽/SCS、调制深度/纹波、时间参数、判定。
- 结果表列：带宽、调制深度、纹波、上升/下降时间、脉冲宽度、判定。
- 本轮只保持当前行为，不扩展业务实现。

## 7. 8.5 ACLR

### 固定参数

- 1466 ARB 时钟：10.0 MHz。
- 带宽：200、400、600、800 kHz。
- 默认每个带宽频点：925 MHz。
- 真实仪表同时在线时执行真实流程，否则使用当前模拟数据模式。

### 运行表

- 列：序号、测试点、配置、测量值、状态。
- 实际运行时行数按带宽与输入频点组合生成。

### 结果表

- 列：序号、测试条件、测量项目、设定值、测量值、合格标准、判定。
- 保留主信道功率、左右 ACLR、截图和统计图关联。

### 生命周期现状

- 使用 `testRunning` 布尔值检查循环。
- 使用 `QTimer::singleShot(0, ...)` 启动主流程。
- 流程内存在 `QThread::msleep()`。
- 停止不能中断正在执行的同步查询或休眠。

## 9. 8.7 ACS

### 固定参数

- ARB 时钟：30.72 MHz。
- 干扰功率：-53 dBm。
- 边缘偏移：100 kHz。
- BLER 门限：10%。
- 自动截图：启用。

### 展示模板

- 参数摘要：
  - 有用信号：200 kHz / 15 kHz DSB。
  - 干扰信号：5 MHz DFT-s-OFDM NR。
- 模板表包含正负 340 kHz、正负 2500 kHz 场景。
- 真实流程使用 upper/lower 候选波形测试点并保存截图。

### 人工输入

- 仪表配置和截图完成后开放 BLER 输入。
- 确认后更新 BLER 与测试点图、PASS/FAIL 饼图及柱状图。
